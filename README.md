# GPRO
An easy to use / install solution heavily based on OpenProject.

## Install
1. Clone the repo.
2. Go into the repo.
3. Check the \[vars\] section on the script to change settings (Like 'path' and 'exposed_port').
4. run ```$ make install```.
5. run ```$ gpro install```.
6. Now whenever you need you can run ```$ gpro run```.
